# mle-razrabotka-v-ide-reshenie
Авторское решение самостоятельного задания спринта 6 курса Инженер Машинного Обучения Яндекс Практикума


Этот проект предназначен для автоматического создания отчетов на основе сырых данных.


## Установка необходимых библиотек:

Для установки необходимых библиотек выполните команду в терминале, предварительно активировав виртуальное окружение Python:

```bash
pip install -r requirements.txt
```


## Запуск:

Для генерации отчета выполните команду в терминале, находясь в корневой папке проекта:

```bash
python3 main.py
```
